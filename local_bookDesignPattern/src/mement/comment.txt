Mementoを一言でいうと、「状態の保存」
ゲームのセーブデータクラスを作る感じ。

セーブデータクラス、セーブデータクラスを管理するクラス、セーブデータを使うクラスに
分ける。

セーブデータを管理するクラスで、
・セーブデータの更新（セーブデータのフィールドの更新）
・セーブデータ時点の巻き戻し（セーブデータのフィールド値を持ってくる）
を行う。


PlantUML:
@startuml
class Main

package game{
class Gamer
class Memento
}

Gamer -down-> Memento:Creates
Main -right-> Gamer:Requests   
Main o- Memento

@enduml
